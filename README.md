# KOVR | Own The Street

![KOVR Banner](https://res.cloudinary.com/dk1tkgjpj/image/upload/v1775499212/Estilo_yzigdi.png)

**KOVR** es una plataforma de comercio electrónico de alta gama dedicada al calzado urbano premium. Diseñada con una estética minimalista y técnica (Techwear), la aplicación ofrece una experiencia de usuario inmersiva centrada en el rendimiento y el estilo.

## 🚀 Tecnologías

Este proyecto ha sido construido utilizando las tecnologías más modernas del ecosistema frontend:

- **React 18**: Biblioteca principal para la interfaz de usuario.
- **TypeScript**: Tipado estático para un código más robusto y mantenible.
- **Tailwind CSS**: Framework de utilidades CSS para un diseño responsivo y sofisticado.
- **Framer Motion**: Motor de animaciones para transiciones fluidas y efectos de parallax.
- **Lucide React**: Set de iconos minimalistas y consistentes.
- **Vite**: Herramienta de construcción ultrarrápida.

## ✨ Características Principales

- **Hero Inmersivo**: Efectos de profundidad y parallax que destacan el producto estrella.
- **Carrusel de Tendencias**: Navegación fluida entre categorías con filtros dinámicos.
- **Modal de Producto Premium**: Detalles técnicos, selector de tallas y visualización de alta resolución.
- **Diseño Responsivo**: Experiencia optimizada para móviles, tablets y escritorio.
- **SEO Optimizado**: Meta tags configurados para una visibilidad máxima en buscadores y redes sociales.

## 🎨 Guía de Estilo

- **Tipografía**: 
  - `Bebas Neue`: Títulos de gran impacto.
  - `Syncopate`: Acentos técnicos y navegación.
  - `Outfit`: Cuerpo de texto y descripciones.
  - `Barlow Condensed`: Precios y datos numéricos.
- **Colores**:
  - `Negro Profundo (#0A0A14)`: Fondo principal.
  - `Rojo Coral (#FF3A2D)`: Color de acento y marca.
  - `Gris Técnico (#111827)`: Contenedores y tarjetas.

## 🛠️ Instalación y Desarrollo

1. Clona el repositorio.
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```
4. Construye para producción:
   ```bash
   npm run build
   ```
